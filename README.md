# Semester Fee Calculator - PUC ALL IN ONE

This is a specialized module for the PUC ALL IN ONE application, designed to help students calculate their semester fees accurately based on academic credits and cost parameters.

## Project Description
The Semester Fee Calculator provides a streamlined interface for students to input their total credits and the specific cost per credit. The system processes the input and generates a digital receipt summary through a seamless Activity transition.

## Functional Features
* **Credit Processing:** Dynamically calculates total tuition fees based on user input.
* **Data Persistence:** Transfers calculated results between Activities using Android Intent Bundles.
* **Input Validation:** Prevents application runtime errors by validating empty or null numeric fields.
* **Material Design:** Utilizes a clean, card-based interface for improved user experience.

## Technical Specifications
* **Programming Language:** Java
* **UI Framework:** XML (Android Layouts)
* **Design Pattern:** Imperative UI with Intent-driven navigation
* **Minimum Android Version:** API 21 (Android 5.0)
